## Tic Tac Toe game  

A simple implementation of the classic Tic-Tac-Toe game in Python.

  
### Features 
  
 **Hot Seat Mode**: Two players can take turns on the same machine.   
   
**Play Against Computer**: One player can play against an automated computer opponent.
Winning and Drawing Conditions: Correctly identifies when a player wins or when the game ends in a draw.
Basic AI for Computer Opponent: The computer opponent makes strategic moves to either win or block the player from winning.
  
  
### How to Play
  
**Setup and Requirements**:

- Ensure you have Python 3.x installed on your machine.  

- Clone the repository to your local machine:   
`git clone https://github.com/Tais19991/Python-Projects/edit/main/Tic_Tac_Toe_game`


### Run the Game:

Run the following command to start the game:  
`python main.py`


### Game Modes:  

**Choose between two game modes**:  
- Hot Seat (1): Play with another player on the same machine.  
- Play with Computer (2): Challenge an automated opponent.


### Gameplay:  

- Follow the prompts to input your moves during your turn.  
- For Hot Seat mode, each player alternates turns inputting their moves.  
- For Play with Computer mode, the computer opponent will automatically make its moves.  

### Winning and Draw:  

- The game will notify you when a player wins or when the game ends in a draw.  
- The board will be displayed after each move to show the current state of the game.
